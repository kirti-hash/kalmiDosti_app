# 🌿 Kalmi Dosti – iOS App (SwiftUI)

Kalmi Dosti is a modern iOS app built using **SwiftUI**, currently focused on creating a clean, smooth, and engaging **UI/UX experience**.  
This repository contains the UI screens and animations that form the foundation of the app’s onboarding and chat experience.

---

## ✨ Features (UI Completed So Far)

### 🟢 1. Splash Screen  
- Full-screen gradient background  
- Clean centered logo  
- Smooth transition to onboarding  


---

### 🟢 2. Onboarding UI (Multi-Step)  
- Modern typography with custom fonts  
- Gradient backgrounds  
- Large centered illustrations  
- Page indicators  
- “Next” button with smooth navigation animations  
- “Skip” button aligned to the right  
- Completely custom layout using **ZStack + VStack**

#### Current Screens:
1. **Confuse About Screen**  
   - App logo  
   - Title text  
   - Subtitle explaining onboarding  
   - Custom page control  
   - “Next” CTA button  
   - Skip option  


---

### 🟢 3. Custom Alerts  
A completely custom-designed modal alert with:
- Dimmed background  
- Tappable outside area to dismiss  
- Only an **OK** button  
- Rounded card design  
- Dark/light adaptable  
- Smooth fade-in animation  
- Reusable across entire project  

---

### 🟢 4. Login UI (In Progress)  
- Clean modern textfields  
- Custom buttons  
- Smooth transition from onboarding  
- Reusable input components  

---

### 🟢 5. Chat Screen (In Progress)  
- Early structure created
